网络安全基础

从 0 到 1 掌握网络安全的核心理念与方法

课程目录

1. CIA 三元组与零信任

网络安全最核心的目标是确保信息的 机密性、完整性 与 可用性（CIA）。 现代企业架构正逐步采纳 零信任 模型：不假设任何内部网络是安全的，所有访问都需要持续验证、最小权限与微分段。

通过加密、访问控制、数据脱敏、密钥管理等手段，避免未授权访问。

利用哈希校验、数字签名、不可变日志等技术，确保数据未被篡改。

冗余、弹性伸缩、抗 DDoS、防灾与演练，确保系统在攻击或故障时仍可提供服务。

2. 常见威胁与攻击链

了解威胁类别与 MITRE ATT&CK 攻击技术图谱可以帮助我们进行威胁建模与防护设计。 常见威胁包括：社工钓鱼、弱口令、漏洞利用、横向移动、数据外泄等。

侦察 → 武器化 → 投送 → 利用 → 安装 → 命令控制 → 目标达成。每一环都可布控探针与阻断点。

日志与可观测性是检测的关键：终端检测响应（EDR）、SIEM、NDR、零信任代理、审计日志等。

3. 安全基线与最小权限

安全基线是企业安全的地板线：补丁管理、配置加固、凭证轮换、密钥分级、备份与演练、最小权限与零信任访问。

  • 身份优先：MFA、短时令牌、基于属性与情境的访问（ABAC/RBAC+），严格区分人机身份
  • 数据优先：分级分域、最少披露、加密端到端、数据水印与追踪
  • 服务优先：服务网格 mTLS、策略中心化、密钥托管与轮换

4. 法律与道德（必读）

本课程仅用于安全教育。请务必遵守当地法律法规与职业道德，仅在授权与隔离的环境下进行测试。任何对生产或第三方系统的未授权测试均属于违法行为。
