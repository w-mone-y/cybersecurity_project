逆向工程

理解二进制结构、静态/动态分析与合规要求

课程目录

1. 环境与法律边界 2. 静态分析：反汇编/反编译 3. 动态分析与沙箱 4. 报告与披露

1. 环境与法律边界

仅在授权样本与隔离环境，遵循负责任披露。严格区分恶意分析与合法研究，避免传播样本。

2. 静态分析：反汇编/反编译

理解指令流与控制流图（CFG）、字符串与导入表，识别混淆与加壳的常见迹象。

3. 动态分析与沙箱

在沙箱/虚拟机中观测 API 调用、网络连接、注册表/文件行为，记录 IOC（Indicators of Compromise）。

4. 报告与披露

输出结构化报告：样本哈希、环境、行为、IOC、处置建议；遵循披露窗口与红线。
